# The experiment results and code of the paper "An Empirical Study of LLMs in Generating Design Rationale for Software Architecture Decisions"

## Introduction
This repository contains the experiment results of the paper *"An Empirical Study of LLMs in Generating Design Rationale for Software Architecture Decisions"*, as well as the implementation code for the LLM-based agents presented in the paper. The dataset and code are provided to help other researchers and developers replicate the experiments or conduct further research.

## Directory Structure
- `Code/`: contains the implementation code of the LLM-based agents.
- `Pilot_Results` provides the results of the pilot experiments.
- `Results/`: provides the experiment results of the three prompting strategies.
- `Collected_Data.xlsx` provides the data used in this study.
